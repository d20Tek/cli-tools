# d20Tek cli tools

## Introduction
A set of command-line tools and utilities to make coding easier. These are .NET tools that can be installed to extent the dotnet command-line experience. 
These command apps are build using [SpectreConsole](https://github.com/spectreconsole/spectre.console) library, which assist in build easy command-line tools.

The set of command-line tools include:
*  [create-guid](./src/D20Tek.Tools.CreateGuid)

For future releases, I will build new, richer tools.
